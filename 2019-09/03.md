## Loadmonitor

目前是掛 瑞翔為 負責人

### 功能介紹

監控加工的過程，
提供 user 這些資訊，
讓 user 去優化這支加工程式

可能根據這些資訊
將 某些參數 去做調整
優化這支加工程式

當然也可以將這些優化的過程 去記錄下來
讓 其他 user 也能迅速地了解 這支加工程式的演變過程
後續追蹤 bug 也很方便

### 資料欠缺

目前已經有 主軸負載 輸出功能
轉速(S command)、 負載 (S load)
但 目前的數值 是抓 Actual

後續須討論是否有要抓其他的數值

ex:

- Feedrate (進給率)
- ...etc (待討論)
